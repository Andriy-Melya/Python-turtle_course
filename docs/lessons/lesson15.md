# 🔁 Цикли і допоміжні змінні

## 🌟 Історія

Черепашка вже навчилася повторювати команди за допомогою циклів.  
Але під час нової експедиції у **Лабораторію алгоритмів** вона помітила дивну проблему 🤔

Кожна нова фігура ставала:  
- більшою 📈  
- зміщувалась убік ↔️  
- або змінювала колір 🎨  

Повторювати однакові команди вже недостатньо…

Тепер їй потрібно навчитися використовувати **допоміжні змінні** — спеціальні змінні, які допомагають:  
- рахувати;  
- змінювати розмір;  
- керувати рухом;  
- створювати складні візерунки.  

---

# 🧠 Що таке допоміжна змінна?

Допоміжна змінна — це змінна, яка змінюється всередині циклу та допомагає керувати малюнком.

Наприклад:

```python
size = 20

for i in range(5):
    forward(size)
    left(90)
    size += 10
```

📌 Тут змінна `size`:

* спочатку дорівнює `20`;
* після кожного повторення збільшується;
* тому лінії стають довшими.

---

# 🔧 Основні ідеї

| Команда             | Що робить                 |
| ------------------- | ------------------------- |
| `x = x + 1`         | збільшує змінну           |
| `x += 1`            | короткий запис збільшення |
| `x -= 1`            | зменшує значення          |
| `for i in range(n)` | цикл з лічильником        |
| `i`                 | номер повторення          |

---

# 🎯 Для чого це потрібно?

Допоміжні змінні дозволяють створювати:

* 🌈 спіралі
* 📏 фігури різного розміру
* 🎆 візерунки
* 🧱 сходинки
* 🌌 складні малюнки

---

# 🚀 Міні-приклад — кольорова спіраль

```python
from turtle import *

speed(0)

size = 10

for i in range(20):
    forward(size)
    left(90)
    size += 5

done()
```

---

# 📌 Що відбудеться?

Після кожного повторення:

* черепашка проходить більшу відстань;
* фігура починає “розкручуватися”;
* утворюється спіраль.

---

# 🎨 Міні-приклад — ряд квадратів

```python
from turtle import *

speed(0)

x = -200

for i in range(5):
    penup()
    goto(x, 0)
    pendown()

    for j in range(4):
        forward(50)
        left(90)

    x += 70

done()
```

---

# 🧩 Що тут важливо?

📌 Змінна `x`:

* зберігає координату;
* після кожного квадрата змінюється;
* тому квадрати не накладаються.

---

# ⚡ Головна ідея теми

Цикл повторює команди.
Допоміжна змінна змінює результат кожного повторення.

Разом вони дозволяють створювати:

* складні алгоритми;
* красиві анімації;
* справжню комп’ютерну графіку 🎨🐢
