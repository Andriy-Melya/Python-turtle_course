## 🎨 Що це за сайт?

Це простий онлайн-інструмент, де ти можеш:

- 🎚️ змінювати значення **Red, Green, Blue**
- 👀 одразу бачити, як змінюється колір
- 📋 отримати код кольору: RGB

---

## 🧪 Як працювати

1. Знайди поля:
   - Red (R)
   - Green (G)
   - Blue (B)
2. Змінюй числа (0–255)
3. Спостерігай:
   - як змінюється колір
   - які значення RGB отримуються

---

## 🎮 Міні-завдання

Спробуй зробити такі кольори:

- 🔴 Червоний → (255, 0, 0)  
- 🟡 Жовтий → (255, 255, 0)  
- 🔵 Синій → (0, 0, 255)  
- ⚪ Білий → (255, 255, 255)
