# ✍️ Виведення тексту в Turtle

## 📝 Чи знаєш ти?

Черепашка може не тільки малювати лінії 🐢  
Вона також вміє **писати текст прямо на екрані!**

---

## 💡 Основна команда

Щоб написати текст, використовується команда:

```python
write("Твій текст")
```

---

## 🔍 Приклад

```python
from turtle import *

write("Привіт!")

done()
```

👉 На екрані з’явиться напис **Привіт!**

---

## 📍 Як розмістити текст

За замовчуванням текст пишеться там, де стоїть черепашка.

Щоб перемістити її:

```python
penup()
goto(x, y)
```

---

### 📌 Приклад

```python
from turtle import *

penup()
goto(-100, 0)

write("Тут мій текст")

done()
```

---

## 🎨 Як змінити вигляд тексту

Можна задати:

* шрифт
* розмір
* стиль

```python
write("Текст", font=("Verdana", 16, "bold"))
```

---

## 🔍 Пояснення

```python
("Verdana", 16, "bold")
```

* `"Verdana"` — назва шрифту
* `16` — розмір
* `"bold"` — жирний текст

---

## 🧪 Приклад з оформленням

```python
from turtle import *

penup()

goto(-120, 20)
write("Вітаю в експедиції!")

goto(-120, -40)
write("Черепашкова регата", font=("Verdana", 16, "bold"))

done()
```

---

## 🔍 Розбір параметра font

У команді:

```python
write("Текст", font=("Verdana", 16, "bold"))
```

використовується спеціальний параметр `font`, який складається з **трьох частин**.

---

## 📊 Таблиця пояснення

| Частина   | Приклад     | Що означає                                           |
| --------- | ----------- | ---------------------------------------------------- |
| 🅰️ Шрифт | `"Verdana"` | Назва шрифту (як виглядають букви)                   |
| 🔢 Розмір | `16`        | Розмір тексту (чим більше число — тим більший текст) |
| 💪 Стиль  | `"bold"`    | Стиль тексту (жирний, звичайний тощо)                |

---

## 🎨 Варіанти значень

### 🅰️ Шрифти

* `"Arial"`
* `"Verdana"`
* `"Courier"`
* `"Times New Roman"`

---

### 🔢 Розмір

* `10` — маленький
* `16` — середній
* `24` — великий

---

### 💪 Стилі

* `"normal"` — звичайний
* `"bold"` — жирний
* `"italic"` — курсив

---

## 🧪 Приклади

```python
write("Привіт", font=("Arial", 12, "normal"))
write("Hello", font=("Courier", 18, "bold"))
write("Hi", font=("Verdana", 20, "italic"))
```

---

!!! tip "Підказка"
Спробуй змінювати тільки один параметр і дивись, що змінюється 👀

```



```python
from turtle import *

penup()
color("red")

write("Червоний текст")

done()
```
