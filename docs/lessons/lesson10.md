# 🎨 Як утворюються кольори в комп’ютері

## 🌈 Чи знаєш ти?

У комп’ютері **немає фарб чи олівців**.  
Усі кольори створюються за допомогою **світла**!

---

## 💡 3 основні кольори

Комп’ютер використовує лише 3 кольори:

- 🔴 **Red (червоний)**
- 🟢 **Green (зелений)**
- 🔵 **Blue (синій)**

!!! info "Це важливо"
    Це називається **RGB**

---

## 🧪 Як утворюються інші кольори?

| Колір | Як отримати |
|------|------------|
| 🟡 Жовтий | червоний + зелений |
| 🟣 Фіолетовий | червоний + синій |
| 🔵 Блакитний | зелений + синій |
| ⚪ Білий | всі разом |
| ⚫ Чорний | нічого |

---

## 🔢 Як це виглядає в коді?

Кожен колір — це **три числа від 0 до 255**:

```python
(червоний, зелений, синій)
```

### Приклади:

```python
(255, 0, 0)   # червоний
(0, 255, 0)   # зелений
(0, 0, 255)   # синій
(255, 255, 0) # жовтий
```

---

## 🐢 У Turtle

Щоб використовувати RGB, потрібно додати:

```python
from turtle import *

colormode(255)
```

Тепер можна писати кольори числами 👇

```python
from turtle import *

colormode(255)

color((255, 0, 0))  # червоний
forward(100)
```

---

## 🎮 Спробуй сам

Зміни числа і подивись, що вийде:

```python
color((100, 200, 50))
```

!!! tip "Підказка" 
    Змінюй кожне число окремо і спостерігай, як змінюється колір 🎨



## ✏️ Колір лінії та 🎨 колір заповнення

У черепашки є **два різні кольори**:

---

### ✏️ Колір лінії

Це колір, яким черепашка **малює контур (лінії)**.

---

### 🎨 Колір заповнення

Це колір, яким **зафарбовується фігура всередині**.

---

## 💡 Як задати обидва кольори одразу

У Turtle можна вказати **два кольори в одній команді**:

```python
color((255, 0, 0), (255, 100, 100))
```

👉 Перший колір — **лінія (контур)**  
👉 Другий колір — **заповнення (всередині)**

---

## 🔍 Пояснення прикладу

```python
color((255, 0, 0), (255, 100, 100))
```

* `(255, 0, 0)` → 🔴 яскраво-червоний контур
* `(255, 100, 100)` → світліший червоний (рожевий) всередині

---

## 🔥 Важливо

Щоб побачити заповнення, потрібно використати:

```python
begin_fill()

# малюємо фігуру

end_fill()
```

---

## ✅ Повний приклад

```python
from turtle import *

colormode(255)

color((255, 0, 0), (255, 100, 100))

begin_fill()

for _ in range(4):
    forward(100)
    right(90)

end_fill()
```

---

!!! tip "Підказка"
    Спробуй змінювати **другий колір** і дивись, як змінюється заливка 🎨
