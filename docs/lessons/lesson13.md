# 🔁 Що таке цикл і для чого він потрібен?

## 🌟 Історія

Після довгих подорожей морем алгоритмів, черепашка 🐢 помітила одну дивну річ…

Вона дуже часто повторює однакові команди:

- вперед
- поворот
- вперед
- поворот...

І так знову і знову 😵

— Це ж можна робити швидше! — вигукнула черепашка.

І тут з’явився **Мудрець Літописець** 📜

> “Якщо дія повторюється — використовуй **цикл**”

---

## 🔁 Що таке цикл?

Цикл — це команда, яка дозволяє **повторити дію кілька разів**.

---

## 💡 Приклад без циклу

```python
from turtle import *

forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)
forward(100)
right(90)

done()
```

😬 Багато однакових команд…

---

## 🚀 Приклад з циклом

```python
from turtle import *

for i in range(4):
    forward(100)
    right(90)

done()
```

🔥 Те саме, але значно коротше!

---

## 🧠 Як працює цикл?

```python
for i in range(4):
```

| Частина    | Що означає         |
| ---------- | ------------------ |
| `for`      | початок циклу      |
| `i`        | змінна (лічильник) |
| `range(4)` | повторити 4 рази   |

---

## 📌 Важливо

!!! info "Запам’ятай"
Все, що написано **з відступом**, буде виконуватись у циклі

---

## 🔷 Малюємо фігури

### 🔺 Трикутник

```python
for i in range(3):
    forward(100)
    right(120)
```

---

### 🔷 Квадрат

```python
for i in range(4):
    forward(100)
    right(90)
```

---

### 🔶 П’ятикутник

```python
for i in range(5):
    forward(80)
    right(72)
```

---

## 🧪 Формула повороту

Щоб намалювати правильну фігуру:

```
кут = 360 / кількість_сторін
```

| Фігура      | Кількість сторін | Кут |
| ----------- | ---------------- | --- |
| Трикутник   | 3                | 120 |
| Квадрат     | 4                | 90  |
| П’ятикутник | 5                | 72  |
